# 类型的计算

## 联合

``` typescript
type A  = string | number;

type Ponit = { x: number, y: number };
type StatusDesc = { state: string };

// type X = Ponint | StatusDesc;
type X = Ponint & StatusDesc; 

const p: X = {x: 1, y: 1, state: 'hello'}; 


interface A {
 foo():void
}

interface A {
 bar():void
}

// A 是 合并

```
