
🤔： 语句和表达式的区别

`Rust` 被称作表达式语言，在 `C` 中，`if` 和 `switch` 是语句，它们不会产生值，也不能在表达式中间使用。在 `Rust` 中，`if` 和 `match` 可以产生值

🤔： `;`分号的代表不同的含义

`;` 代表语句，不带则代表表达式

🤔： 控制流： `if` 和 `match`

### if let 

let 使用的是模式匹配
``` rust
// pattern 模式
if let pattern = expr {  
     block1  
} else {  
    block2  
}
```
`match` 的语法糖

🤔： 循环流

``` rust
while condition {  
    block  
}  
  
while let pattern = expr {  
    block  
}  
  
loop {  
    block  
}  
  
for pattern in itertable {  
    block  
}
```
