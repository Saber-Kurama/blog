使用hexo搭建一个博客

几年前曾经创建一个github pages的博客。可惜后来没写过文章。如今从新弄一下。记录下自己的搭建的过程。
参考的是使用 github Pages 服务建立 ixirong.com 独立博客全过程

  1. 去github上clone下hexo，或者查看hexo.io中文文档按照文档执行

  2. 选择皮肤
    参照 hexo你的博客
    参照 如何使用 Jacman 主题
    个人选择了 pacman 的皮肤 （分类和标签的信息来源于文章中的categories 和 tags）
    我只是简单的增加了分类和标签菜单

  3. 添加新文章
    执行$ hexo new '文章名字'
  4. 本地运行
    执行$ hexo server
  5. 打包并部署到github上
    参照部署文档
    在_config.yml中配置
    repository最好配置SSH
    执行命令

本篇只是简单记录自己创建的过程。详细请参照我创建的过程中参照的。关于个性化，以及自定义主题。以后有时间再研究。
